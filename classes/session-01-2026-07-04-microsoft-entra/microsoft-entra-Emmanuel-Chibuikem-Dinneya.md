## Key Concepts I Learned

<!-- Write the main ideas covered in today's session -->

- Microsoft Entra ID is a microsoft engine that manages  authetication (identity) and authorization grants access to Microsoft services such as Microsoft 365, Azure, Power BI etc.
- Aunthentication Methods and Authentication Strength: there are various authentication methods which are Passwordless Method, Multifactor Methods and Single-factor method..of which the passwordless method is the strongest.
- MFA Enforcement Using Conditional Access Policy: Conditional Access is a security strategy that decides based on signals, whether to allow, block or strenghten a sign-in or user. It is more like an "IF-THEN" workflow.

- Privileged Identity Management is used for users that can do administrative tasks so that users do not necessarily have a permanent role assignmemt.
